Can you decode the following string dGg0dF93NHNfczFtcEwz from base64 format to
ASCII?

To decode base64 Linux has a base64 command. To decrypt using it you need to
feed it a string using a file, or by piping it in with echo, and the -d flag.

command: echo "dGg0dF93NHNfczFtcEwz" | base64 -d

Answer: picoCTF{th4t_w4s_s1mpL3}
